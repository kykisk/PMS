### 동작 설명
- **사이드바**: 전체 모듈 네비게이션 (현재 위치 하이라이트)
- **상단 액션 바**: 4가지 입력 방식 버튼 + 템플릿 다운로드 버튼
- **필터 바**: 검색 + 분류/상태/우선순위/입력경로 드롭다운 필터
- **테이블**: 요구사항 목록 (정렬 가능한 컬럼 헤더)
- **작업 컬럼**: 수정(✏️), 삭제(🗑) — 행 클릭 시 상세화면 이동
- **버전 저장**: 현재 요구사항 리스트 전체를 스냅샷
- **내보내기**: Excel/PDF 선택 다운로드
- **LLM 미설정 시**: MD Import, 기술서 Import 버튼 비활성화 + 툴팁

### 동작 설명
- **단일 스크롤**: 기본정보 → 연결항목 → 버전이력 순서로 섹션 구분
- **연결 항목 섹션**: C안 하이브리드 + 경유 항목 펼쳐서 표시
  - 기능 하위에 Task 트리로 표시
  - 테스트는 "직접 연결" vs "기능 경유" 구분
  - [+ 연결] 버튼으로 새 링크 추가, ❌ 로 해제
  - AI 생성 버튼 인라인 배치
- **버전 이력 섹션**: 버전 목록 + 비교/복원 액션

- **결과 컬럼**: 테스트 수행 결과 (Pass/Fail/미수행)
- **내보내기**: Excel/PDF 선택 다운로드

## 공통 레이아웃 원칙

### 상세화면 섹션 구성 (전 화면 공통)

모든 상세화면은 **탭 없이 섹션 구분**으로 한 화면에서 스크롤하며 확인:

| 상세화면 | 섹션 순서 |
|---------|----------|
| 요구사항 | 기본정보 → 연결항목 → 버전이력 |
| 기능 | 기본정보 → 화면설계서 → 연결항목 → 버전이력 |
| Task | 기본정보 → 연결항목 → 이슈/리스크 → 버전이력 |
| 테스트 시나리오 | 기본정보 → 테스트케이스 목록 → 연결항목 → 버전이력 |
| 테스트 케이스 | 기본정보 → 수행결과 → 연결항목 → 버전이력 |
