## 실행 방법 (공통)

```bash
# 1. DB 시작
podman start pms_db_dev

# 2. BE 실행 (터미널 1)
cd /home/ec2-user/workspace/PMS/backend
DATABASE_URL="postgresql://pms_user:pms_password@localhost:5432/pms_db" node dist/src/main.js

# 3. FE 개발 서버 (터미널 2)
cd /home/ec2-user/workspace/PMS/frontend
npm run dev
# → http://localhost:5173

# Swagger
# → http://localhost:3000/api/docs
```
